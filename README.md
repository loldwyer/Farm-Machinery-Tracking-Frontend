# Farm Machinery Tracker Front-End

A responsive front-end web application for tracking and managing farm/factory machinery information. This project was completed as part of a group project, where I was responsible for the front-end development, including the HTML, CSS and client-side JavaScript implementation.

The application provides a clean user interface for viewing machinery records, checking machine status, and supporting admin-style management of machinery data. It was designed to work with a backend system/API, while the front-end focuses on usability, layout, navigation and displaying machinery information clearly.

## Project Overview

The Farm Machinery Tracker was developed to help manage machinery records in a structured and accessible way. The system is intended for environments where machines need to be monitored, updated and maintained, such as manufacturing, factory or agricultural settings.

My role in the group project focused on building the front-end interface and ensuring that the pages were responsive, user-friendly and suitable for integration with backend functionality.

## How to Run This Project

This front-end can be opened locally in a browser or run using a local development server.

### Method 1: Open Locally

1. Download or clone the repository.
2. Open `index.html` in a modern web browser.
3. Navigate through the available pages using the menu.

### Method 2: Use VS Code Live Server

1. Open the project folder in Visual Studio Code.
2. Install the **Live Server** extension.
3. Right-click `index.html`.
4. Select **Open with Live Server**.
5. The website will open in your browser.
